Clear Cache - Absen App

Gunakan tools ini untuk membersihkan cache dan memperbaiki masalah login.

Langkah-langkah:

  1. Klik "Clear All Caches" untuk membersihkan semua cache
  2. Klik "Unregister Service Worker" untuk menghapus service worker lama
  3. Klik "Reload Page" untuk memuat ulang halaman
  4. Buka aplikasi utama di index.php
